## Завдання 1 — get_days_from_today

Створіть функцію `get_days_from_today(date)`, яка розраховує кількість днів між
заданою датою та поточною датою.

### Вимоги
- Вхідний параметр `date` — рядок у форматі `YYYY-MM-DD`
- Повертає ціле число (кількість днів)
- Якщо дата у майбутньому — результат від’ємний
- У розрахунках враховуються лише дні
- Використовується модуль `datetime`

### Приклад використання

```python
get_days_from_today("2026-02-10")
```
Якщо сьогодні 2026-02-08, результат буде:
-2

## Завдання 2 — get_numbers_ticket

Функція `get_numbers_ticket(min, max, quantity)` генерує набір унікальних випадкових
чисел для лотереї.

### Вимоги
- `min` ≥ 1
- `max` ≤ 1000
- `quantity` — кількість чисел у діапазоні
- Результат — відсортований список унікальних чисел
- Якщо параметри некоректні — повертається порожній список

### Приклад використання

```python
lottery_numbers = get_numbers_ticket(1, 49, 6)
print(lottery_numbers)
```
Можливий результат:
```
[4, 15, 23, 28, 37, 45]
```

## Завдання 3 — normalize_phone

Функція `normalize_phone(phone_number)` нормалізує телефонні номери до стандартного
формату для SMS-розсилки.

### Вимоги
- Видаляє всі символи, крім цифр та `+`
- Якщо міжнародний код відсутній — додає `+38`
- Повертає номер у вигляді рядка

### Приклад використання

```python
raw_numbers = [
    "067\\t123 4567",
    "(095) 234-5678\\n",
    "+380 44 123 4567",
]

sanitized_numbers = [normalize_phone(num) for num in raw_numbers]
print(sanitized_numbers)
```
Результат:
```
['+380671234567', '+380952345678', '+380441234567']
```

## Завдання 4 — get_upcoming_birthdays

Функція `get_upcoming_birthdays(users)` формує список привітань на найближчі 7 днів
(включаючи поточний день).

### Вимоги
- Вхідні дані — список словників з ключами `name` та `birthday`
- Дата народження у форматі `YYYY.MM.DD`
- Якщо день народження припадає на вихідний — привітання переноситься на понеділок
- Результат — список словників з ключами:
  - `name`
  - `congratulation_date` (формат `YYYY.MM.DD`)

### Приклад використання

```python
users = [
    {"name": "John Doe", "birthday": "1985.02.10"},
    {"name": "Jane Smith", "birthday": "1990.02.14"},
]

upcoming_birthdays = get_upcoming_birthdays(users)
print(upcoming_birthdays)
```
Можливий результат (якщо сьогодні 2026.02.08):
```
[
    {'name': 'John Doe', 'congratulation_date': '2026.02.10'},
    {'name': 'Jane Smith', 'congratulation_date': '2026.02.16'}
]
```
