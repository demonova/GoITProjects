## [goit-pycore-hw-03](goit-pycore-hw-03) Python Functions

Містить рішення для чотирьох завдань з роботи з датами, випадковими числами та обробкою рядків у Python.

Реалізовано функції:
1. [`get_days_from_today(date)`](goit-pycore-hw-03/src/task1.py) - розраховує кількість днів між заданою датою та поточною датою.

2. [`get_numbers_ticket(min, max, quantity)`](goit-pycore-hw-03/src/task2.py) - генерує набір унікальних випадкових чисел для лотереї.

3. [`normalize_phone(phone_number)`](goit-pycore-hw-03/src/task3.py) - нормалізує телефонні номери до стандартного формату для SMS-розсилки.

4. [`get_upcoming_birthdays(users)`](goit-pycore-hw-03/src/task4.py) - формує список привітань на найближчі 7 днів (включаючи поточний день).

## [goit-pycore-hw-04](goit-pycore-hw-04) CLI & File Processing

Містить реалізацію чотирьох завдань, пов’язаних з:
   - роботою з файлами
   - обробкою текстових даних
   - рекурсивним обходом директорій
   - створенням CLI-застосунків

Реалізовано:
1. [`total_salary(path)`](goit-pycore-hw-04/task1.py) - функція аналізує текстовий файл із зарплатами розробників та повертає:
    - загальну суму зарплат
    - середню зарплату

2. [`get_cats_info(path)`](goit-pycore-hw-04/task2.py) - функція читає файл з інформацією про котів та повертає список словників.

3. [Візуалізація структури директорії (CLI)](goit-pycore-hw-04/task3.py) - скрипт приймає шлях до директорії як аргумент командного рядка та виводить структуру файлів і папок.

4. [Консольний бот-помічник](goit-pycore-hw-04/task4.py) - CLI-застосунок для роботи з контактами, які зберігаються у словнику:
    - ключ — ім’я
    - значення — номер телефону

## [goit-pycore-hw-05](goit-pycore-hw-05) Python Assistant Bot & Core Concepts 

Демонструє ключові концепції Python:
   - Closures (Замикання)
   - Memoization (Кешування)
   - Recursion (Рекурсія)
   - Generators (Генератори)
   - Regular Expressions (Регулярні вирази)
   - Decorators (Декоратори)
   - Exception Handling (Обробка винятків)
   - Clean Code Architecture

Реалізовано:
1. [`caching_fibonacci()`](goit-pycore-hw-05/task1.py) - функція кешування Фібоначчі, яка:
   - Використовує рекурсію
   - Кешує результати
   - Повертає внутрішню функцію (closure)
   - Має складність O(n)

2. [Генератор та підрахунок прибутку](goit-pycore-hw-05/task2.py):
   - Витягує дійсні числа з тексту
   - Використовує yield
   - Підсумовує значення через функцію вищого порядку

4. [Консольний бот з декоратором](goit-pycore-hw-05/task4.py):
   - Дороблено консольного бота з попереднього домашнього завдання
   - Централізувано обробку помилок через декоратор
   - Перехоплює ValueError, KeyError, IndexError
   - Не завершує програму при помилках
